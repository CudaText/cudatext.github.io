Text Converters

There is plugin "Text Converter" (install it via Addons Manager) which allows to encode text by replacing text fragments with another fragments. Set of converters in plugin menu can be customized. Open folder "Data\conv" inside SynWrite install folder, you'll see there "txt" files. Each file defines two converter menu items: "encode" item converts text in forward direction and "decode" item converts back. "Decode" item is not shown, if you add "_" char in filename just before extension.

Addons Manager has special kind of add-ons for this plugin. Names of these add-ons start with string "Text converter:".

Format of txt-files

  • Encoding: ANSI or UTF-16 (with BOM).
  • Lines are processed if they're not empty, and don't start with single ";" char.
  • Lines define substring convertion in one of the forms:
    • StrFrom StrTo - two strings separated with a space, all StrFrom occurences will be replaced with StrTo.
    • ;;{StrFrom} StrTo - first substring is enclosed into ;;{ }, it's needed if it contains spaces or starts with ";".
