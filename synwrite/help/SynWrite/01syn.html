SynWrite - Overview

Editor features

  • Syntax highlighting for lots of languages
  • Highlightings are fully customizable (in visual editor)
  • Code folding
  • Support for almost all encodings
  • Multiple carets and selections
  • Panels:
    • Tree structure view for source code
    • File explorer
    • Project management
    • FTP/SFTP client
    • Document mini-map
    • Clipboard history
    • External tools output
    • Search results
    • Text clips
    • Tabs list
  • Coding helpers:
    • Auto-completion
    • Auto-closing of tags
    • Text snippets
    • SyncEditing feature
    • SmartTagTabbing feature
    • Emmet (Zen Coding) support
    • Color preview + Color picker
    • Images preview
    • Inserting date/time stamp
    • Portable bookmarks
    • Column markers
    • Micro-map
    • HTML Tidy
  • Search and replace with regular expressions
  • Search and replace in multiple files
  • External tools support (capture of console output, errors navigation)
  • Extraction of strings mathing RegExp
  • Customizable hotkeys
  • Bookmarks
  • Keyboard macros
  • Spell checking
  • File sessions
  • Plugin-version for "Total Commander" program
  • Multi-lingual user interface
  • Export to RTF/HTML with syntax highlighting
  • Portable mode
  • and more...
