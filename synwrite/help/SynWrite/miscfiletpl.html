New File Templates

You can customize list of templates, which is suggested by "File -- New" menu. To do it, call command "File -- New -- Browse templates folder" and create a new file in this folder.

The extension of your file will determine lexer for the new document (you must use file extension already known by some lexer). The name of your file will determine the menu item caption. For example, if your template name is "custom.xml", then new menu item will be called "File -- New -- X -- XML custom".

If you need to set "UTF8, no BOM" encoding for the new document, name your template file with "_UTF8" text at the filename end (before extension). If "UTF8 with BOM" encoding is needed, just save the template as UTF8.
