Integration with TotalCmd search

One commonly asked question is how to use TotalCmd's search string while SynWrite Lister plugin is called from TotalCmd search results panel (Alt+F7). This is possible, use one of the following methods:

  • turn on SynWrite option "TcHistory" (see below)
    (note: this option works only with TC 8.0+)
or:
  • turn off SynWrite option "SynDialog" (see below)
  • call search not using Ctrl-F/F3 but using Lister's menu
or:
  • turn off SynWrite option "SynDialog" (see below)
  • in SynWrite options assign Ctrl-F/F3 to commands "Lister find"/"Lister find next"

Hidden options

This is a list of hidden options of SynWrite Lister plugin. Options are read from LSPlugin.ini file (it's located in the same folder as your wincmd.ini file), from [Syn2] section.

SynDialog=N This turns off (N=0) or on (N=1) flag, which allows SynWrite to show its own custom Search/Replace dialog, instead of default Lister dialog.
TcHistory=N This turns off (N=0) or on (N=1) flag, which allows SynWrite to use Total Commander ini file directly, section [SearchText] inside it. When on, SynWrite will store its search strings history directly into TotalCmd ini file.
QViewToolbars=Item1,Item2,... This allows to show some toolbars/panels in QuickView mode. Possible values of items are:
  • Menu
  • File
  • Edit
  • View
  • QS
  • Tree
  • Clip
  • Output
  • User1
  • User2
  • User3
QViewTree=ext1,ext2,... This allows to show Tree panel in QuickView mode for specified file extensions only. This key, if not empty, overrides the QViewToolbars key.
QViewRO=N This turns off (N=0) or on (N=1) "Read-only" mode for QuickView.
RO=N This turns off (N=0) or on (N=1) "Read-only" mode for non-QuickView.

Note: QuickView is a special TotalCmd mode, which is activated by Ctrl+Q key in TotalCmd file panels.
