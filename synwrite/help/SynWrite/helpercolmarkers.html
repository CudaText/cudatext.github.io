Column Markers

There is plugin "Column Markers" which gives several commands to control column markers: set markers, jump to lefter marker, jump to righter marker.

Dialog "Set markers": you need to enter space-separated list of column positions (don't enter right-margin position). To remove all, just enter an empty string.
