Source Code

SynWrite application is open source. You can find Delphi 7 source code in the Readme subfolder of installation folder, Src.rar archive. But note: it uses shareware component EControl Syntax Editor. Also optionally shareware spell-checker component Addict 3 can be used.

It uses several open source components not included in Src.rar:

  • Toolbar2000 (2.2.2)
  • SpTBXLib (2.4.8)
  • Png Delphi (1.564)
  • Png Components (1.0 RC3)
  • DKLang (3.01)
  • Python for Delphi
  • TPerlRegEx (pcre-8.31)
  • Tnt Unicode Controls (2.3.0)
  • Fundamentals (SQL Parser, 4.0, modified cStrings)
  • FastCode (0.6.4)
  • FastMM (4.78)
  • SynTaskDialog (modified for Widestring)
  • LibTar (2.1.2)

Note: original components' code must be tweaked a little for SynWrite: see help file "SourceCodeNotes.chm" in the Readme folder.

For "Explorer" panel plugin the following is needed:

  • VirtualTreeview
  • VirtualShellTools
  • EasyListview
