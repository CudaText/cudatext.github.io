Dialog "Lorem Ipsum Generator"

This dialog allows to insert into text so called "Lorem Ipsum" text - several nonsence lines (example: Lorem ipsum dolor sit amet.), which may be used to present HTML pages. You can use such options:

  • "Sentences": insert single long line; "Count" field is number of sentences in a line;
  • "Paragraphs": insert several long lines; "Count" field is number of paragraphs;
  • "Use HTML tags": format paragraphs by <p> tags (otherwise it's plain text separated with double newlines).
