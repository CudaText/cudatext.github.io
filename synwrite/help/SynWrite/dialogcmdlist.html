Introduction

These two dialogs share the similar look. They consist of items list-box and filter edit-box. You can move selection (even when filter box is focused) with keys Up, Down, PageUp, PageDown, Ctrl+Home, Ctrl+End. Filter box allows to show only those items which match the filter string.


Normal and fuzzy search

If checkbox "Fuzzy search" is checked at the bottom, then filter box will use "fuzzy" algorithm: only those items will be left, which contain all filter letters in ascending order. E.g.:

  • "fsav" mathes "File: Save file"
  • "fsav" matches "Focusing: Toggle master/slave focus"

If "Fuzzy search" is unchecked, then only those items will be left, which contain all the words from the filter string. E.g.:

  • "fi re" matches "File: Reopen file"
  • "fi re ti" matches "Run: Tidy - configure"

Project files list: File/folder names

Only in the "Project files list" dialog:
You can type into filter box two strings separated with "\" char - first of these strings will match the directory path, and second one will match the file name. E.g.:

  • (normal search) "mypro\mycla" matches "D:\Test\MyProject\myclass.py"
  • (normal search) "\my" or "my" matches files with "My" substring in name (not path)
  • (normal search) "my\" matches files with "My" substring in path (not name)
  • (fuzzy search) "te\my\my.py" matches "D:\Test\MyProject\myclass.py"

Project files list: Content filter

Only in the "Project files list" dialog:
You can add to filter box (at the end) text @SomeName, to search for text SomeName inside files. Only those files will be left which contain text specified after "@" character.

Search is case-sensitive if one "@" char is typed, and case-insensitive if "@@" chars are typed before a text. Search can find partial words (not "whole words only").
