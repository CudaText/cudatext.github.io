Customizing Status Line

To customize content of statusbar 1st panel (which shows current line and column numbers), you need to manually edit the following options in the Syn.ini file (it is located in "%AppData%\SynWrite" folder by default), in section [View]:

StatusNoSel contains macros list used when no selection is made
StatusSmallSel contains macros list for single-line selection
StatusStreamSel contains macros list for stream selection (more than 1 line selected)
StatusColumnSel contains macros list for column selection
StatusCarets contains macros list for multi-carets mode
StatusWidth specifies panel width, in pixels
StatusAlign specifies text alignment: 0: center, 1: left, 2: right

The defaults are: 

[View]
StatusNoSel={LineNum} : {ColNum} ({TotalLines})
StatusSmallSel={LineNum} : {ColNum} ({SelLines}x{SelCols}/{TotalLines})
StatusStreamSel={LineNum} : {ColNum} ({SelLines}/{TotalLines})
StatusColumnSel={LineNum} : {ColNum} ({SelLines}x{SelCols}/{TotalLines})
StatusCarets=#={Carets} ({TotalLines})
StatusWidth=150
StatusAlign=0

The following macros are allowed in options values:

{LineNum} Caret's line number
{ColNum} Caret's column number
{TotalLines} Total lines count
{TotalChars} Total characters count
{SelLines} Number of selected lines
{SelCols} Number of selected columns, for column blocks
{SelChars} Number of selected characters
{Carets} Multi-carets mode: number of carets
{CaretsTopLine} Multi-carets mode: line number for top caret
{CaretsBottomLine} Multi-carets mode: line number for bottom caret
{FileDate} Date of disk file, long format
{FileDate2} Date of disk file, short format
{FileDateOp} Date of disk file, format using option "Date/time format"
