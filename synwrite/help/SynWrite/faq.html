How to import/export lexers.

You can export/import lexers to/from text files. Such files are written in "Delphi resources" format and have extension "lcf".

1st method:

  • Activate needed lexer (in Lexer menu)
  • Call menu command "Options -- Customize lexer"
  • In the opened "Lexer properties" dialog press Export or Import button
  • (Note: import will overwrite selected lexer from chosen file)

Alternative method:

  • (No need to select lexer first)
  • Call menu command "Options -- Customize lexer library"
  • In the opened "Lexer library" dialog click needed lexer, then press "Import lexer" or "Export lexer" toolbar button
  • (Note: import will not overwrite selected lexer, new lexer will be created)

To speed-up program start:

Open "Options -- Customize lexer library" dialog, and delete (not just disable!) unneeded lexers. Then save the lexer library.

After you have customized the lexer library, save its copy under different name: old name is "LexLib.lxl", new name may be "MyLib.lxl" (it's recommended to place it into the same folder, so "Syn.ini" key will not contain path). You can create such copy using "Save lexer library with another name" button in the "Lexer library" dialog. After that, make sure you activated your new library: press "Choose another lexer library" button in the "Lexer library" dialog, and browse for your new file.

You should also turn off the following options in the Options dialog:

  • "Spell Check" tab -- "Enable live spell cheking"
  • "History" tab -- "Remove non-existing items from recents list at startup"

To speed-up editor redrawing:

Turn off these options:

  • "View Settings" tab -- "Show horizontal ruler"
  • "Select/Highlight" tab -- "Highlight pair brackets"

How to make app portable / How to copy settings to another PC.

Since last versions, app is always portable - settings are stored in ini-files in "Settings" subfolder of program folder. (File "portable.ini" not needed.)
Note: do not install app to "Program Files", default install-folder is "C:\SynWrite".

How to adjust program colors.

To change SynWrite interface colors:

  • call "Options" dialog, "Colors" tab
  • change the toolbar theme and/or interface items' colors

To change colors of lexers' elements:

  • call needed lexer in Lexer menu
  • call command "Options -- Customize lexer styles"
  • for needed items change their colors/fonts

To open two files side-by-side:

Use the Groups of tabs feature: set grouping mode to "2 horizontally", then drag any tab from 1st group to 2nd one.
If needed, you may use toolbar commands "Sync vertical scrolling"/"Sync horizontal scrolling" to synchronize scrolling of active tabs in groups 1 and 2.

To split view of a single file:

Right-click tab header and call popup menu command "Split this tab". You may also split tab horizontally, using splitter's popup menu.

How to keep lexers settings on app reinstalling.

You can keep your own lexer settings, using one of the following:

  • If you just need to keep customized lexer's styles: you can reinstall SynWrite and then perform command "Options -- Advanced -- Restore lexer styles". This command restores your custom styles from SynStyles.ini file, which is auto-saved on every OK button press in Lexer properties dialog.

  • If you need to keep entire lexer config (including internal lexer tweaks): you can export your custom lexer to a file, and after SynWrite reinstalling you can import it back. See Export/Import topic.

  • If you have created new lexer library file and activated it (using "Choose another lexer library" button in the "Lexer library" dialog), then don't worry about reinstalling - your custom library file won't be changed.

Is there a "Column Mode" command like in Notepad++.

Yes, it's command "Edit -- Edit column". It places multi-selections for selected column block, so you may type new block content. After finished with typing, do a simple click out of block, to cancel multi-carets mode.

Note: "Edit column" has no special shortcut, you can use shortcut for other command "Edit -- Multi-carets -- Carets from selection, left edge", which is used internally for "Edit column" command.

What monospaced fonts are good for program.

Default font "Consolas" looks good in Windows Vista/7/8. But you may try other monospaced fonts for use in SynWrite:

  • Lucida Console
  • Liberation Mono

Lucida Console is part of Windows. To add Liberation font, download ttf package and install "LiberationMono-Regular.ttf".

Note: for better readability it's suggested to set option "Line spacing" to value 1 or 2.

How to disable "indentation guides".

To hide indentation guides (vertical lines, which show syntax blocks such as "{...}" in C language), you need to use option "View Settings -- Block staples".

How to hide red underlines for misspelled HTML tags.

  • Activate HTML lexer
  • Call dialog "Options -- Customize lexer" (for HTML lexer)
  • Click "Styles" tab of dialog; you'll see two styles "Tag id - underlines", "Tag attrib - underlines"; click them and change their color/underline styles on the right side of dialog
  • Save lexer library when prompted

You can keep your styles on SynWrite reinstalling - see another question.

You may need to do the same for "PHP_dev_HTML" lexer (HTML sublexer of PHP(dev) lexer). To call properties of this lexer, go into "Options -- Customize lexer library" dialog.

How to create SynWrite translation.

  • Download DKLang Translation Editor
  • In Translation Editor open "Readme\Syn.dklang" file. Create your new LNG file.
  • Put your LNG file into "Lang" subfolder of SynWrite folder, select it in Options dialog and test it.
  • Send your file to contact email (About box).

Program name.

In march 2010, there was a usual Total Commander plugin named "Syn2". Name "Syn2" appeared from famous Delphi component named SynEdit. "2" meaned that plugin was rewritten (by first author Alexander Ryabov) using another Delphi component, EControl.

Syn2 was forked by me, Alexey, and our first version was "Syn2 v2.1". After some time we began to search for new name, and Matthias (sourceforge project admin) gave few variants: SynPad, SynText, SynWrite etc etc... and we have chosen "SynWrite".

How to change font size for specific lexer.

Open "Lexer library" dialog (command "Options -- Customize lexer library"), select needed lexer and double-click it to open lexer properties dialog. In the lexer properties dialog create "Default" style, if it's not created yet:

  • activate tab "Styles" and create new style,
  • rename style to "Default" using edit-field below the styles list,
  • for this new style set "Style type" = "Custom font",
  • button "Set custom font" is enabled now, press it and choose needed font.

You have a style with custom font attached. Now activate dialog tab "Common" and set option "Default style" to this style.

How to assign lexers/shortcuts to command plugins.

First, call menu command "Options -- Advanced -- Edit file SynPlugins.ini". In file SynPlugins.ini find section [Commands] and needed plugin line. Example:

JS Format\Format=py:syn_js_format;run;JavaScript,JSON;Ctrl+Alt+F;

After "=" char are ";"-separated values.
Lexers list (","-separated) is 3rd value (in example: JavaScript, JSON).
Shortcut is 4th value (in example: Ctrl+Alt+F).
You can also assign key-combination like [Ctrl+Q, Q, B] - separate key-combo with "|" char: Ctrl+Q|Q|B
