Commenting

There're several commands to comment/uncomment text blocks (menu "Edit -- Comments"):

Command Meaning
Toggle block comment For selected lines: add (if not commented) or remove (if commented) line-comment chars - at lines beginning.
Toggle block comment (alt.) The same, but place comment chars at first non-space chars position.
Block comment For selected lines: add line-comment chars (at lines beginning).
Block uncomment For selected lines: remove line-comment chars.
Toggle stream comment For selection (can be any, even part of line): add or remove stream-comment chars (at block start and end).
  • Line-comment chars are set in lexer properties dialog (first tab of dialog).
  • Stream-comment chars are set in file Lexers.cfg (it's overwritten on installation) or file LexersEx.cfg (create it by yourself, same format), in the folder of SynWrite installation, in section [Comments].
