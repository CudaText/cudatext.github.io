Markers

In the "Bookmarks" menu you will find "Markers" submenu with markers-related commands. Markers are small red triangles which show positions in the text (while bookmark shows entire line with a color and a gutter icon, marker shows only triangle below the single position). Commands are:

  • Drop marker - Add a marker on caret position (triangle may be barely visible because caret is above it).
  • Collect marker - Jump to last placed marker, and delete this marker (works like a "stack", removes stack item).
  • Swap marker - Jump to last placed marker, delete this marker, but add new marker on previous caret position.
  • Go to last marker - Jump to last placed marker without deleting it.
  • Clear markers - Remove all markers in current file.

Markers are per-file, so e.g. "Collect" takes marker only inside the current file.
