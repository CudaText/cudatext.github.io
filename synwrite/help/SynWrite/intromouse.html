Mouse Usage

Common shortcuts:

  • Right button click - Show context menu.
  • Left button click - Move caret to mouse position and start normal selection.
  • Middle button click - Start "auto-scroll" feature: circle at clicked position is shown; if you move mouse above/below that circle than editor will scroll up/down automatically.

  • Shift + Left click - Select "normal" block from previous caret position to mouse position.
  • Alt + Shift + Left click - Select "column" block from previous caret position to mouse position.
  • Ctrl + Alt + Shift + Left click - Select "line" block from previous caret position to mouse position.

Multi-carets shortcuts:

  • Ctrl + Left click - Add caret.
  • Ctrl + Shift + Left click - Add several carets aligned in a column.

Misc shortcuts:

  • Ctrl + Left click, over an URL - Open URL in default browser.
  • Ctrl + Alt + Left click - Perform "Go to definition" command (only with plugins installed).
  • Shift + rotate mouse wheel - Perform horizontal scrolling.

Dragging operations:

  • Alt + dragging - Perform "column" selection.
  • Ctrl + Alt + dragging - Perform "line" selection.
  • Left button click over selection block - start text dragging. Dragged text will be moved to new position. To copy selection using drag&drop press Ctrl (or Ctrl+Alt) key during dragging.
  • You can perform drag&drop on block staples. Press Ctrl+Alt keys and drag block staple. This operation will indent/unindent lines that are coupled by the block staple.
  • You can drag right margin to new position by left mouse button with pressed Ctrl+Alt keys.

Used terms:

  • "Block staples" are dashed vertical lines, which connect syntax-blocks edges. E.g. in C++ lexer syntax-blocks are braces {...}, in Pascal lexer they are "begin...end".
  • "Right margin" is solid vertical line at configurable column, e.g. column 80.
