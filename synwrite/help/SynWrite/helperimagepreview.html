Images Preview

This feature allows to preview graphic files (jpg/gif/png/bmp/ico), which are referenced in HTML tags, for example:

<img src="images/untitled.jpg" alt="untitled" />

When you hold mouse over image filename, graphic hint appears.
If filename isn't found on local PC, hint will show "???" text.

Notes:

  • Preview of jpg/gif/ico images requires GDI+ library. This library is installed by default on Windows XP and newer systems.
  • Filenames which are web links are not previewed.
