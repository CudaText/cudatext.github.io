Console

Console panel allows to run/test embedded Python-engine commands. To call this panel, use menu command "Windows -- Panels -- Console panel" (default hotkey: Ctrl+tilde). Type commands into bottom edit-field, and finish them with Enter. Commands with their output will be appended to the log field (it's read-only).

  • You can enter command "-" to clear log field.
  • You can enter commands with first "=" char: command "=something" will execute "print(something)".
  • You can double-click log field lines which begin with ">>>" - this will repeat clicked command.
