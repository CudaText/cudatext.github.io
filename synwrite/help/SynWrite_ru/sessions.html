Сессии

Сессии представляют собой состояния программы, которые записываются в файлы с расширением "synw-session". Меню управления сессиями ("Открыть", "Сохранить", "Сохранить как" и т.п.) вы найдете в главном меню в "Файл -- Сессии". Если какая-то сессия открыта, то её имя отображается в заголовке окна SynWrite в фигурных скобках.

Сессии хранят следующие данные:

  • Режим группировки
  • Позиция разделителя групп (для режимов 2х1, 1х2, 1+2)
  • Cвойства вкладок (только для именованных вкладок):
    • имя файла
    • номер группы
    • номер верхней строки редактора
    • позиция каретки
    • состояние "только для чтения"
    • состояние "перенос слов"
    • состояние "разрешено сворачивание кода"
    • состояние "показаны номера строк"
    • режим выделения мышью (нормальный/строками/колонками)
    • позиция разделителя подчинённого редактора (вкладка может быть разделена на 2 редактора: основной и подчинённый)
    • свёрнутые диапазоны текста
    • позиции маркеров колонок
    • цвет ярлыка вкладки

Умолчательная сессия, с именем (default), используется тогда, когда задано автосохранение сессий и именованная сессия не была выбрана. Файл умолчательной сессии сохраняется рядом с файлом "Syn.ini" (по умолчанию в папке %Appdata%\SynWrite).

Чтобы открывать файлы сессий из Проводника Windows двойным кликом, в диалоге по команде меню "Опции -- Интеграция с Проводником" включите соотв-ю опцию.

Команда "Открыть/добавить сессию" делает следующее: она добавляет вкладки из сессии к текущим открытым вкладками.
