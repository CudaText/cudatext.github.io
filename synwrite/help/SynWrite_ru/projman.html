Введение

Вызвать менеджер проекта в SynWrite можно так:

  • открыть панель с синтаксическим деревом и перейти на закладку "Проект";
  • использовать команду "Окна -- Панели -- Панель проекта".

Менеджер проекта позволяет сгруппировать файлы в виде дерева, которое будет на диске храниться в одном файле (будут сохраняться только ссылки на файлы, но не сами файлы). Файлы проекта имеют расширение "synw-proj".

На панели менеджера проекта часть команд предствалена в виде кнопок:

  • Открыть файл проекта
  • Сохранить файл проекта
  • Добавить пустую (виртуальную) папку
  • Добавить содержимое реальной папки
  • Добавить файл(ы) с диска
  • Переименовать текущий элемент
  • Удалить выделенные элементы
  • Открыть выделенные элементы в редакторе
  • Открыть диалог "Свойства проекта"

Проектные файлы можно сделать переносимыми так: поместите все файлы с данными в подпапках какой-то одной папки и сохраните проектный файл в этой же папке. В таком случае эта папка станет "переносимой" - вы сможете переместить ее и воспользоваться проектным файлом без правок путей.


Перетаскивание в дереве проекта

Для узлов дерева допускаются следующие перетаскивания:

  • перенос узла(ов) на папку (обычный) - это перемещение элементов в указанную папку;
  • перенос узла(ов) на папку (с удержанием Alt) - это перемещение элементов в место над указанной папкой (на ее же уровне);
  • перенос узла(ов) на файл - это перемещение элементов в место над указанным файлом (на его же уровне).

Можно перетаскивать файлы и папки из Проводника Windows в дерево проекта. Элементы будут добавлены в проект (с учетом текущего положения в дереве). Если перетаскивается папка, то появится диалог "Добавить папку в проект".


Панель предпросмотра

Можно вызывать/прятать панель предпросмотра текущего файла в проекте - нажатием Пробела, когда менеджер проекта имеет фокус. Панель показывает файл во временном редакторе, в состоянии "только чтение". Можно подвинуть панель и пристыковать ее к краю окна SynWrite. Используйте кнопку "Открыть в редакторе" на панели, чтобы открыть просматриваемый файл для редактирования.
