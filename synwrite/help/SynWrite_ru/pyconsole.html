Консоль

Панель консоли позволяет выполнить любые команды встроенного Python-движка через их ручной ввод. Чтобы вызвать панель, вызовите команду верхнего меню "Окна -- Панели -- Панель консоли" (умолчательное нажатие: Ctrl+тильда). Вводите команды в поле ввода внизу панели и нажимайте Enter. Вывод команд будет добавляться к полю журнала консоли (поле только для чтения).

  • Можно вводить один знак "минус" для очистки консоли.
  • Можно вводить команду с первым знаком "равно" - команда "=текст" будет запускать "print(текст)".
  • Можно делать двойной щелчок в журнале консоли в строках ">>>команда" - это повторит указанную команду.
