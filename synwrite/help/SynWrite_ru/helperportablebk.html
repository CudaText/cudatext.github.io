Портабельные закладки

"Портабельные закладки" - это текстовые комментарии специального назначения, SynWrite рассматривает их как закладки. Естественно, что перемещения или переименования файла на таких закладках не сказываются (в отличии от "обычных" закладок, которые теряются в этих случаях).

Эти закладки помещают в комментарий текст, начинающийся с "NOTE" или "TODO".

пример для HTML <!--NOTE: текст -->
пример для CSS /*NOTE: текст */
пример для PHP //TODO: текст
пример для файла без лексера (*NOTE: текст *)

Для работы с этими закладками есть две команды в меню "Закладки": "Добавить портабельную закладку" и "Перейти к портабельной закладке...".

Замечания:

  • Можно вручную сформировать такую закладку (или даже создать для этого сниппет). Пробелы перед комментарием несущественны.
  • Можно выключить отображение (*NOTE при показе диалога "Список закладок", для этого есть скрытая опция.
