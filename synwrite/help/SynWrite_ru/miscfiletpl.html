Шаблоны новых документов

Можно дополнять список шаблонов, который появляется в меню "Файл -- Создать". Для этого вызовите команду "Файл -- Создать -- Просмотр папки шаблонов" и создайте новый файл в той папке.

Расширение вашего файла определит лексер для нового документа (вы должны использовать только те расширения, которые знакомы хотя бы одному лексеру). Имя файла будет использовано в элементе меню. Например, если файл назвать "custom.xml", то элемент меню будет выглядеть как "Файл -- Создать -- X -- XML custom".

Если для нового документа вы хотите использовать кодировку "UTF8, no BOM", завершите имя файла текстом "_UTF8" (до расширения). Если нужна кодировка "UTF8 with BOM", то просто сохраните файл с такой кодировкой.
