Исходный код

У программы SynWrite открытый исходный код. Исходный код на Delphi 7 находится в подпапке Readme папки, где установлен SynWrite, в виде архива Src.rar. Только учтите: нужен платный компонент EControl Syntax Editor SDK. Опционально можно использовать платный spell-checker Addict 3.

Используются следующие свободные компоненты, они не включены в Src.rar:

  • Toolbar2000 (2.2.2)
  • SpTBXLib (2.4.8)
  • Png Delphi (1.564)
  • Png Components (1.0 RC3)
  • DKLang (3.01)
  • Python for Delphi
  • TPerlRegEx (pcre-8.31)
  • Tnt Unicode Controls (2.3.0)
  • Fundamentals (SQL Parser, 4.0, изменен cStrings)
  • FastCode (0.6.4)
  • FastMM (4.78)
  • SynTaskDialog (изменен для Widestring)
  • LibTar (2.1.2)

Исходный код некоторых из этих компонент должен быть немного изменен для применения в SynWrite: смотрите "SourceCodeNotes.chm" из папки Readme.

Для сборки плагина "Explorer" нужны:

  • VirtualTreeview
  • VirtualShellTools
  • EasyListview
