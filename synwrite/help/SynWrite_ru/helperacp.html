Дополнительные возможности

Есть дополнительные функции, которые можно включить/выключить в диалоге "Опции" на закладке "Автозавершение".

  • Дополнять список словами из текущего файла. Эта фунция доступна для всех лексеров, в то время как обычное автозавершение доступно только для некоторых из них.
  • Можно настроить так, чтобы список для автозавершения появлялся сам после набора некоторого количества символов в слове.

Общий формат файлов

Файл общего формата для работы автозавершения - это текстовый файл в ANSI кодировке, содержащий строки следующей структуры: 

Type Id |Hint
Type Id (Param1; Param2; ...) |Hint
Type Id (Param1; Param2; ...): ResultType |Hint

Слово на месте "Type" будет отображаться в первой колонке локального списка, слово на месте "Id" - во второй колонке, слово на месте "Hint" - в третьей. Список параметров в скобках будет использоваться для фунции "Подсказка параметров".
Для разделения параметров можно использовать как ";", так и ",".
Конструкцию "|Hint" можно опускать.
Символы "\" в части "Hint" надо экранировать: "\\".
Пробелы в части "Id" надо записывать как "%20" (такое допустимо для всех символов диапазона 0x20..0x2F).

Первая строка файла может быть "контрольной": в ней указывается, какие символы могут встречаться в словах: #chars символы
Например, если в слове могут встречаться символы минус, точка, "#" и "$", контрольная строка будет такой: 

#chars .-#$
