Формат даты/времени

Ниже приведен список описателей, которые задают конкретный вид строки для команды "Правка -- Вставить дату/время". Используйте эти описатели при задании опции "Формат времени/даты" в диалоге настроек.

Конструкция Отображение

c

Отображает дату в коротком формате и после нее время в длинном формате. Время не отображается, если значение указывает на точную полночь.

d

Отображает день в виде числа без начальных нулей (1-31).

dd

Отображает день в виде числа с начальными нулями (01-31).

ddd

Отображает день недели в сокращенном виде (Пн-Вс).

dddd

Отображает день недели в полном виде (понедельник-воскресенье).

ddddd

Отображает дату в коротком формате (31.12.2001).

dddddd

Отображает дату в длинном формате (31 Января 2001 г.).

e

Отображает год, как число без начальных нулей в текущем периоде/эре (только для локализаций Японии, Кореи и Тайваня).

ee

Отображает год, как число с начальными нулями в текущем периоде/эре (только для локализаций Японии, Кореи и Тайваня).

g

Отображает краткое обозначение периода/эры (только для локализаций Японии и Тайваня).

gg

Отображает полное обозначение периода/эры (только для локализаций Японии и Тайваня).

m

Отображает месяц в виде числа без начальных нулей (1-12). Если конструкция m находится непосредственно после конструкции h или hh, то отображаются минуты, а не месяц.

mm

Отображает месяц в виде числа с начальными нулями (01-12). Если конструкция m находится непосредственно после конструкции h или hh, то отображаются минуты, а не месяц.

mmm

Отображает месяц в сокращенном виде (янв-дек).

mmmm

Отображает месяц в полном виде (Январь-Декабрь).

yy

Отображает год в виде двух цифр (00-99).

yyyy

Отображает год в виде четырех цифр (0000-9999).

h

Отображает час в виде числа без начальных нулей (0-23).

hh

Отображает час в виде числа с начальными нулями (00-23).

n

Отображает минуты в виде числа без начальных нулей (0-59).

nn

Отображает минуты в виде числа с начальными нулями (00-59).

s

Отображает секунды в виде числа без начальных нулей (0-59).

ss

Отображает секунды в виде числа с начальными нулями (00-59).

z

Отображает милисекунды в виде числа без начальных нулей (0-999).

zzz

Отображает милисекунды в виде числа с начальными нулями (000-999).

t

Отображает время в коротком формате (23:59).

tt

Отображает время в длинном формате (23:59:59).

am/pm

Указывает на 12-часовое отображение времени для предыдущей конструкции h или hh. Отображает 'am' для часов ранее полдня и 'pm' для часов после полдня. Конструкция не зависит от регистра.

a/p

Указывает на 12-часовое отображение времени для предыдущей конструкции h или hh. Отображает 'a' для часов ранее полдня и 'p' для часов после полдня. Конструкция не зависит от регистра.

ampm

Указывает на 12-часовое отображение времени для предыдущей конструкции h или hh.

/

Отображает разделитеть полей в дате.

:

Отображает разделитеть полей во времени.

'xx'/"xx"

Символы в одиночных или двойных кавычках отображаются "как есть" и не влияют на остальное форматирование.
