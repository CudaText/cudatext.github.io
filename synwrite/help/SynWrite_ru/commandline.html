Параметры командной строки

Запускаемый файл "Syn.exe" можно запустить такими способами:

  • Syn
  • Syn "файл1" [ "файл2" ... ] [ опции ]
  • Syn "папка1" [ "папка2" ... ] [ опции ]
  • Syn "/two=файл1|файл2|строка1|строка2|колонка1|колонка2|отступ"
  • Syn "/cmp=файл1|файл2"
  • Syn /reg

При запуске без параметров программа откроет либо пустое окно, либо последнюю сессию/проект.
При запуске со списком файловых имен программа откроет файлы во вкладках.
Можно передавать не имена файлов, а имена папок - в этом случае будут открыты все текстовые файлы из указанных папок и всех их подпапок.

При запуске с ключём "/two" программа откроет два файла, в двух группах, каждый файл будет прокручен до указанной строки/колонки. Номера строк/колонок необязательны. "Отступ" задает вертикальный отступ каретки от края редактора (минимум 0).
При запуске с ключём "/cmp" программа вызовет плагин сравнения двух файлов (если такой плагин установлен).
При запуске с ключём "/reg" программа покажет только диалог "Интеграция с Проводником".

Опциональные ключи командной строки:

  • /ro - Открывать файл(ы) в режиме "только чтение".
  • /s - Включить опцию "Разрешать запуск только одной копии программы".
  • /n=номер - Перейти на указанный номер строки после открытия файла(ов).
