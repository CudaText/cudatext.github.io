HTML Tidy

HTML Tidy - это утилита, которая помогает искать ошибки в HTML-документах и выполнять некоторые действия с HTML-страницами: "перевод тегов в нижний регистр", "очистка документа" и другие.

Утилита доступна как плагин (ставится через "Менеджер дополнений"). Команды плагина:

  • "Меню": дает вызвать команды Tidy. Команды настраиваются через текстовые файлы.
  • "Валидация документа": в панель "Валидация" (рядом с "Результатами поиска") помещаются сообщения об ошибках и предупрежденях в HTML-коде. Двойной клик на строке сообщения позволяет перейти к нужной строке кода.
  • "Просмотр конфигураций": показывает список текстовых файлов для команд Tidy, можно править файлы, добавлять новые.
