Использование мыши

Обычные сочетания:

  • Клик правой кнопкой - Показ контекстного меню.
  • Клик левой кнопкой - Каретка перемещается к курсору. Если не отпускать кнопку, то начинается обычное выделение.
  • Клик средней кнопкой - Начинается авто-скроллирование: появляется круговой маркер, при перемещении курсора выше/ниже маркера происходит прокрутка вверх/вниз.

  • Клик левой кнопкой с Shift - Выделяет "обычный" блок от предыдущей позиции каретки до кликнутой позиции.
  • Клик левой кнопкой с Alt + Shift - Выделяет "вертикальный" блок от предыдущей позиции каретки до кликнутой позиции.
  • Клик левой кнопкой с Ctrl + Alt + Shift - Выделяет "строчный" блок от предыдущей позиции каретки до кликнутой позиции.

Сочетания для работы с каретками:

  • Клик левой кнопкой с Ctrl - Добавление новой каретки.
  • Клик левой кнопкой с Ctrl + Shift - Расстановка кареток вдоль колонки.

Специальные сочетания:

  • Клик левой кнопкой над ссылкой с Ctrl - Открытие ссылки в умолчательном браузере.
  • Клик левой кнопкой с Ctrl + Alt - Выполнение команды "Перейти к объявлению" (только с установленными плагинами).
  • Поворот колеса с Shift - Горизонтальная прокрутка.

Перетаскивание:

  • Протяжка с Alt - Выполняет "вертикальное" выделение.
  • Протяжка c Ctrl + Alt - Выполняет "строчное" выделение.
  • Можно перетаскивать выделенный блок текста. Для копирования вместо перемещения нужно удерживать Ctrl (или Ctrl+Alt).
  • Можно менять отступ у синтаксического блока, перетаскивая с Ctrl и Alt скрепку блока, которая указывает на этот блок.
  • Можно перетаскивать указатель "правой границы", если удерживать Ctrl и Alt.

Использованные термины:

  • "Скрепка блока" - это вертикальная линия, соединяющая концы синтаксического блока. Например, для лексера C++ это текст с фигурными скобками {...}, в лексере Pascal это конструкция "begin...end".
  • "Правая граница" - это вертикальная линия, обозначающая некоторую колонку, например 80, и настраиваемая в диалоге опциий (вкладка "Настройки вида").
