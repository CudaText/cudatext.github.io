Проверка правописания с учетом синтаксиса

Проверка правописания в SynWrite учитывает синтаксис исходного кода. При этом проверяются только слова из комментариев и строковых констант. Например, для кода на языке C комментарии имеют вид /* Какой-то текст */, а строковые константы заключены в двойные кавычки: "Какой-то текст".

Как настроить проверку с учетом синтаксиса для новых лексеров?

Для настройки своего лексера, нужно внести изменения в файл "Lexers.cfg" (в папке, где установлен SynWrite). Нужно указать специальные стили для вашего лексера: стиль комментариев в секции [CommentStyles] и стиль строковых констант в секции [StringStyles]. Можно указать несколько стилей через запятую.

Замечание: файл "Lexers.cfg" обновляется при каждой инсталляции SynWrite. Поэтому лучше внести ваши строки в файл "LexersEx.cfg" (создайте его в той же папке).
