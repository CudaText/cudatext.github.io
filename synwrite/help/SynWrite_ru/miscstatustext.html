Настройка статус-строки

Для настройки данных, отображаемых в первом поле статус-строки (там по умолчанию стоят текущая строка и колонка), нужно самостоятельно отредактировать в файле Syn.ini (см. папку "%AppData%\SynWrite") в секции [View] следующие ключи:

StatusNoSel данные для состояния без выделения
StatusSmallSel данные для состояния с выделением в пределах одной строки
StatusStreamSel данные для состояния с обычным выделением, захватывающим несколько строк
StatusColumnSel данные для состояния с вертикальным выделением, захватывающим несколько строк
StatusCarets данные для состояния с несколькими каретками
StatusWidth указание ширины поля в пикселах
StatusAlign указание выравнивания текста: 0: по центру, 1: прижать влево, 2: прижать вправо

Умолчательные значения этих ключей: 

[View]
StatusNoSel={LineNum} : {ColNum} ({TotalLines})
StatusSmallSel={LineNum} : {ColNum} ({SelLines}x{SelCols}/{TotalLines})
StatusStreamSel={LineNum} : {ColNum} ({SelLines}/{TotalLines})
StatusColumnSel={LineNum} : {ColNum} ({SelLines}x{SelCols}/{TotalLines})
StatusCarets=#={Carets} ({TotalLines})
StatusWidth=150
StatusAlign=0

В значениях этих ключей могут встречаться следующие макропеременные:

{LineNum} Номер строки с кареткой
{ColNum} Номер колонки с кареткой
{TotalLines} Полное число строк
{TotalChars} Полное число символов
{SelLines} Число выделенных строк
{SelCols} Число выделенных колонок при вертикальном выделении
{SelChars} Число выделенных символов
{Carets} При нескольких каретках: число кареток
{CaretsTopLine} При нескольких каретках: номер строки с верхней кареткой
{CaretsBottomLine} При нескольких каретках: номер строки с нижней кареткой
{FileDate} Дата файла на диске в длинном формате
{FileDate2} Дата файла на диске в коротком формате
{FileDateOp} Дата файла на диске в формате, описанном в опции "Формат времени/даты" в диалоге "Опции"
