Как импортировать/экспортировать лексер?

Вы можете импортировать/экспортировать лексеры в/из текстовые(х) файлы. Данные в них хранятся в формате "ресурсы Дельфи", а файлы имеют расширение "LCF".
Первый способ:

  • Активируйте нужный лексер (через меню Лексер)
  • Выполните команду меню "Опции -- Изменение свойств лексера"
  • В диалоге "Настройки лексера" нажмите на Экспорт или Импорт
  • (Замечание: импорт необратимо обновит лексер из выбранного файла)

Другой способ:

  • (Не нужно сначала выбирать лексер)
  • Вызовите команду меню "Опции -- Изменение библиотеки лексеров"
  • В диалоге "Библиотека синтаксических анализаторов (лексеров)" укажите нужный лексер и нажмите в тулбаре на кнопку "Импорт лексера" или на кнопку "Экспорт лексера"
  • (Замечание: импорт не обновит выбранный лексер, а создаст новый)

Как ускорить работу программы?

Чтобы ускорить запуск программы:

Через меню "Опции -- Изменение библиотеки лексеров" откройте диалог и удалите (недостаточно отключить!) лишние лексеры. После этого сохраните библиотеку лексеров.

Совет. После изменений библиотеки лексеров сохраните ее под новым именем (чтобы инсталляция не перезаписала файл): если исходное имя было "LexLib.lxl", новое может быть "MyLib.lxl" (рекомендуется сохранять в той же папке). Можно создать копию кнопкой "Сохранить библиотеку лексеров под другим именем". После этого вы сможете перейти к работе с этой библиотекой кнопкой "Выбрать другую библиотеку лексеров".

Отключите следующие опции:

  • вкладка "Проверка орфографии" - "Разрешить авто-подсветку неверных слов"
  • вкладка "История" - "Удалять несуществующие файлы из списка недавних, при запуске"

Чтобы ускорить отрисовку:

Отключите следующие опции:

  • вкладка "Настройки вида" - "Показывать линейку"
  • вкладка "Выделение/Подсветка" - "Подсвечивать парные скобки"
  • вкладка "Выделение/Подсветка" - "Подсвечивать все вхождения *******" (2 опции)

Как сделать переносимую сборку? / Как перенести настройки на другой ПК?

Теперь программа всегда портабельна - настройки хранятся в файлах в подпапке Settings папки программы. (Файл portable.ini не нужен.) Только не ставьте программу в папку "Program Files", чтобы не было проблем с записью.

Как настроить цвета?

Для изменения цвета у элементов интерфейса SynWrite:

  • в диалоге "Опции" откройте закладку "Цвета"
  • измените тему для тулбара и/или цвета элементов интерфейса
Для изменения цвета у элементов лексера:
  • установите нужный лексер через меню Лексер
  • вызовите команду меню "Опции -- Изменение стилей лексера"
  • измените цвет/шрифт для нужных элементов

Как увидеть текст из двух файлов сразу?

Как увидеть текст из двух файлов сразу?
Используйте функцию Группы вкладок: установите режим группировки "2 горизонтально", затем перетащите мышью любую вкладку с первой группы на вторую.
Если нужно, можно использовать кнопки панели инструментов "Синхронность верт. прокрутки"/"Синхронность гориз. прокрутки" для одновременной прокрутки файлов в группах 1 и 2.

Чтобы получить два отображения одного файла:
Кликните правой кнопкой мыши на его ярлыке и в локальном меню вызовите команду "Разделить эту вкладку". Можно разделять вкладку горизонтально или вертикально, использую локальное меню над разделяющей линией.

Как сохранить свои настройки лексера при переустановке/обновлении программы?

Можно восстановить индивидуальные настройки лексера так:

  • Если нужно восстановить только стили: после переустановки SynWrite выполните команду меню "Опции -- Расширенные -- Восстановить стили лексеров...". Стили будут восстановлены из файла SynStyles.ini, куда они автоматически помещаются при каждом нажании на ОК в диалоге свойств лексера.

  • Если нужно восстановить лексер целиком: заранее экспортируйте лексер, а после переустановки SynWrite импортируйте его обратно. Подробности в Как импортировать/экспортировать лексер?.

  • Если вы создали новую библиотеку лексеров и работаете с ней (кнопка "Выбрать другую библиотеку лексеров" в диалоге "Библиотека лексеров"), то вам не нужно беспокоиться при переустановках - ваша библиотека не изменится.

Есть ли в программе режим "Column Mode" как в Notepad++?

Да, есть команда "Правка -- Править колонку". Выделите вертикальный блок и вызовите эту команду - вы сможете печатать новое содержимое всех строк блока. По завершении, сделайте простой клик вне блока, для выключения режима мульти-кареток.

Примечание: команда "Править колонку" не имеет специальной клавиши, но можно использовать клавишу команды "Правка -- Каретки -- Каретки из выделения, по левому краю", т.к. эта команда делает то же самое.

Какие моноширинные шрифты лучше использовать?

Умолчательный шрифт "Consolas" хорошо выглядит в Windows Vista/7/8. Можете также попробовать другие шрифты:

  • Lucida Console
  • Liberation Mono

Lucida Console поставляется с Windows. Для добавления в систему шрифта Liberation скачайте ttf пакет и установите "LiberationMono-Regular.ttf".

Примечание: для улучшения читабельности можно установить опцию "Межстрочный интервал" равной 1 или 2.

Как отключить "indentation guides"?

Чтобы отключить "indentation guides" (вертикальные линии, обозначающие синтаксические блоки), используйте опцию "Настройки вида -- Скобки блоков".

Как отключить красное подчеркивание для неопознанных HTML-тегов?

  • Активируйте лексер HTML
  • Вызовите диалог по команде меню "Опции -- Изменение свойств лексера"
  • Перейдите на вкладку "Стиль". Найдите в левом списке два стиля "Tag id - underlines", "Tag attrib - underlines"; кликните их и измените их свойства (красный цвет и подчеркивание) в правой части диалога
  • Сохраните библиотеку лексеров

Чтобы сохранить измененные стили на будущее, смотрите этот вопрос.

Возможно потребуется сделать то же самое для подлексера "PHP_dev_HTML" в лексере "PHP".

Как создать перевод для интерфейса программы?

  • Скачайте DKLang Translation Editor
  • В Translation Editor откройте файл "Readme\Syn.dklang". Создайте свой LNG-файл.
  • Поместите свой LNG-файл в подпапку "Lang" папки программы, выберите его в диалоге Опции и оттестируйте.
  • Пришлите свой файл на адрес, указанный в диалоге "О программе".

Имя программы.

В марте 2010г существовал обычный плагин к Total Commander с именем "Syn2". Имя "Syn2" - от названия известного Delphi-компонента SynEdit. "2" значило, что плагин был переписан (первым автором Александром Рябовым) на другой движок - EControl.

Syn2 был форкнут мной, Алексеем, и образовался "Syn2 v2.1". Когда я начал искать пристойное имя программе, Matthias (админ проекта на sourceforge) предложил несколько вариантов: SynPad, SynText, SynWrite и т.д... из них выбрали SynWrite.


Как задать размер шрифта для конкретного лексера.

Откройте диалог "Библиотека лексеров" (команда в меню "Опции"). Выберите нужный лексер, сделайте двойной щелчок и откроется диалог свойств лексера. В диалоге св-в лексера создайте стиль "Default", если его еще нет:

  • во вкладке "Стили" создайте новый стиль,
  • переименуйте его в "Default", используя поле ввода под списком стилей,
  • для этого стиля задайте опцию "Тип стиля" = "Заданный шрифт",
  • кнопка "Задать шрифт" теперь разрешена, кликните и задайте шрифт.

У вас теперь есть стиль с заданным шрифтом. Активируйте вкладку "Общие" и для опции "Стиль по умолчанию" задайте ваш стиль "Default".

Как назначить лексеры/клавиши командным плагинам.

Вызовите команду "Опции -- Расширенные -- Открыть файл SynPlugins.ini". В файле SynPlugins.ini найдите секцию [Commands] и строку плагина, пример:

JS Format\Format=py:syn_js_format;run;JavaScript,JSON;Ctrl+Alt+F;

После символа "=" стоят значения через ";".
Список лексеров (разделение по ",") это 3-е значение (в примере: JavaScript, JSON).
Нажатие это 4-е значение (в примере: Ctrl+Alt+F).
Можно задать и комбинацию нажатий типа [Ctrl+Q, Q, B] - записывайте комбинацию через "|": Ctrl+Q|Q|B
