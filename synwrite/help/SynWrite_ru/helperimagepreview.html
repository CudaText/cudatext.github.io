Предпросмотр изображений

Есть возможность предпросмотра графических файлов (jpg/gif/png/bmp/ico), на которые есть ссылки в HTML коде, например, такие:

<img src="images/untitled.jpg" alt="untitled" />

Когда курсор задерживается над именем файла, появляется графическая подсказка.
Если файл не находится, то в подсказке отображается текст "???".

Замечания:

  • Предпросмотр изображений в форматах jpg/gif/ico требует наличия библиотеки GDI+, которая входит в Windows, начиная с Windows XP.
  • Если ссылка указывает на сетевой ресурс, то предпросмотр не выполняется.
