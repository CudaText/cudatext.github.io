Диалог "Свойства проекта"

Этот диалог появляется при нажатии на кнопку "шестерёнка" на панели "Менеджер проекта". В нём изменяются следующие свойства текущего проекта. Все они хранятся в файле проекта.

Кодировка / Окончания строк / Лексер Свойства для новых документов, созданных когда проект загружен. Эти свойства, если для них заданы неумолчательные значения, перекрывают подобные установки в основных настройках SynWrite.
Рабочая папка Папка, с которой начинается работа в диалогах по командам "Добавить файл" и "Добавить содержимое реальной папки".
Эта папка представлена макропеременной {ProjectWorkDir} при вызове внешних инструментов.
Главный файл Полный путь к главному файлу проекта. Единственный способ его задания - в локальном меню командой "Сделать главным файлом".
Это файл представлен макропеременной {ProjectMainFileName} при вызове внешних инструментов.
Сортировка файлов Режим сортировки файлов, полученных командой "Добавить содержимое реальной папки". Если сортировка отключена, эта команда значительно быстрее обработает большую папку. Замечание: можно в любой момент отсортировать элементы любой папки с помощью команд локального меню.
Пути поиска Список папок, которые будут просматриваться командой "Открыть" из локального меню редактора. Этот список дополняет тот, который указан в главном диалоге "Опции".
Переменные Построчно в формате VarName=текст задаются имена и значения пользовательских макропеременных, которые могут применяться в настройках внешних инструментов в виде {VarName}. Нельзя использовать символ ";" в этих объявлениях.
Строки вида AC[ИмяЛексера]=c:\path\MyCompletions.acp позволяют указать дополнительные списки автозавершения к лексерам. Путь к файлу можно не указывать, если он в той же папке, где файл проекта.
