Введение

Emmet - это плагин, который ставится через "Менеджер дополнений". Он дает сокращения для ускоренного набора кода HTML и CSS. Его просто освоить и получить значительную экономию времени. Доступны несколько наборов сокращений: HTML (по умолчанию), CSS (для лексеров CSS, LESS, SCSS, SASS, Stylus и для HTML-лексеров, если каретка находится внутри style-блоков), XSL (для лексеров XML, XSLT).

Вот несколько примеров, чтобы составить впечатление о работе Emmet.

Пример 1

Исходный текст в редакторе: p*3
Можно расширить это сокращение командой Emmet, оно заменится на код: 
<p></p>
<p></p>
<p></p>

Подобным образом можно размножать почти любое из сокращений.

Пример 2

Исходный текст в редакторе: table>tr*2>td
После расширения получится: 
<table>
   <tr>
     <td></td>
   </tr>
   <tr>
     <td></td>
   </tr>
</table>

Пример 3

Исходный текст в редакторе: select>option#item-$*3
После расширения получится: 
<select>
    <option id="item-1"></option>
    <option id="item-2"></option>
    <option id="item-3"></option>
</select>

Профили

Профиль - это стиль конечных выражений, влияет на оформление пустых элементов (<br> или <br /> или <br/>), расстановку дополнительных пробелов, сдвигов и концов строк. Можно изменить активный профиль командой плагина. Доступные профили:

  • "html" — умолчательный профиль.
  • "xhtml" — такой же как "html", но пустые элементы оформляются так: <br />.
  • "xml" — умолчание для файлов XML и XSL: каждый тег размещается на новой строке с подходящим сдвигом, пустые элементы оформляются так: <br/>.
  • "xml_zen" — такой же как "xml", но в максимально вложенные узлы добавляются пустые строки (например, в table>tr>td).
  • "line" — применяется для раскрытия сокращений без разбиения строк и сдвигов. Пустые элементы оформляются так: <br />.
  • "plain" — такой же как "line", но без перемещения каретки внутрь (только для сложных конструкций).
