    Platform-specific notes

    • Windows: Python engine is included.
    • Linux Qt5/Qt6 build: you need libqt5pas/libqt6pas.
    • Linux/Unix: Python engine is not included. To use plugins, see instruction inside "default.json" config about "pylib".
      For Debian/Ubuntu, you need .deb package, it adds CudaText shortcut to the Applications list, and adds "cudatext" to system path. For others, get .xz package, unpack "cudatext" directory to home directory, and create shortcut by hands.
    • Fedora Linux packages: wiki page.
    • macOS:
      • unofficial builds.
      • Python engine is not included. To use plugins, you must install Python from python.org.
      • macOS AArch64 version needs this to be run: 
        xattr -cr /Applications/CudaText.app/
    • Haiku: unofficial builds.
